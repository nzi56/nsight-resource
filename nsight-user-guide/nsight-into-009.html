Map Printing

    • nSight provides a one-click Print Map function that generates a professional, print-ready PDF of the current map view directly from the map toolbar.
    • The exported PDF includes:
      • A clean title page with map title, date/time, user name, and optional custom notes
      • The exact map extent currently displayed, rendered at high resolution
      • Essential cartographic elements automatically added:
        • Accurate scale bar
        • North arrow
        • Grid/reference coordinates (when enabled)
      • A dedicated legend/summary page containing clear, organized legends for all visible layers
    • Users can customize the title, add project references, or adjust layout orientation (portrait/landscape) before export.
    • The resulting PDF is immediately available for download, sharing, or printing — perfect for field reports, stakeholder presentations, regulatory submissions, or operational documentation.

    This feature ensures consistent, high-quality map outputs without requiring external GIS software or manual layout preparation.
