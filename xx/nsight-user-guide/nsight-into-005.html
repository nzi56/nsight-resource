House Hoover Functionality

  • nSight offers an intuitive and efficient mouse-hover (tooltip) inspection capability for rapid feature identification without clicking.
  • Users activate the hover tool by clicking the dedicated Hover Inspection button in the toolbar.
  • Once enabled, moving the mouse over any map feature instantly displays a clean, non-intrusive hover tooltip window containing key attributes and metadata of the feature directly under the cursor.
  • From within the hover tooltip, a Trace Network button allows immediate upstream and/or downstream electrical tracing of the hovered feature, highlighting the entire connected circuit with distinctive animated visual effects.
  • Additionally, the Layer Highlighting & Monitoring control panel provides real-time visual emphasis options:
    • Select any layer or specific monitoring condition from the dropdown menu
    • The system instantly applies a distinct visual style (e.g., glowing outline, color overlay, pulsing symbols) to all features belonging to the selected layer or matching the chosen condition

These hover and dynamic highlighting tools significantly speed up network monitoring, fault diagnosis, and operational awareness—especially useful during outages or when performing quick visual checks across large territories.
